### 使用CocoaPods

在使用前需要安装CocoaPods到你的电脑上，打开终端执行：

```
gem install cocoapods
```

项目中创建Podfile，添加如下内容：

```  
use_frameworks!

platform :ios, '9.0'

target 'Your_Target' do
# UBiX 天璇SDK
	pod 'UBiXMerakSDK'
end
```

更新Podfile后终端执行：

```
pod install 或者 pod update
```
